# **資格取得奨励手当**

資格取得奨励手当にはkickflowへ資格取得申請時のみ支給される①お祝い金と、kickflowで資格取得申請後に毎月支給される②手当の2つの制度がある。  
2019年7月から以下の通りの資格を取得したものに対して、手当を毎月支給する。  
資格を複数種類取得した場合は資格毎に支給するため、合算した金額の手当を支給する。

# ①お祝い金

１．お祝い金は各試験の初回合格時のみ支給する。  
　　※更新時、入社前に取得した資格は適用対象外  
２．各資格に対するお祝い金の支給金額は下表の通りとする。  
３．Google Cloud 資格のみ、各資格毎に資格取得申請の早いメンバー順に5名まで、下表の支給金額とは別にお祝い金5万円を支給するものとする。  
　　ただし、①Professional Cloud Architect②Professional Data Engineer③Associate CloudEngineerの3つの資格については本項の対象外とする。  
４．各月20日までに申請された資格取得申請に係るお祝い金は翌月25日に支給される。  
　　各月21日以降に申請された資格取得申請に係るお祝い金は翌々月25日に支給される。  
５．申請とは、原則、不備のない申請のことを指す。  
　　本人過失の不備（合格証明書の添付忘れ等）がある申請は、申請とはみなさない。  
　　本人過失のない不備（Google側の問題等）の申請は、申請とみなす。

１項については2019年6月以前より変更ないものとする。  
２、３項については2019年8月9日支給給与から適用開始とする。  
４項については2019年7月9日支給給与から適用開始とする。  
５項については2023年11月24日支給給与から適用開始とする。

# ②手当

１．各資格に対する手当の支給金額は下表の通りとするが、支給される手当の上限金額は10万円とする。  
２．手当は各月末日までに申請された資格取得申請に対して翌々月25日から支給開始となる。  
３．申請とは、原則、不備のない申請のことを指す。  
　　本人過失の不備（合格証明書の添付忘れ等）がある申請は、申請とはみなさない。  
　　本人過失のない不備（Google側の問題等）の申請は、申請とみなす。  
この場合、合格者名・合格した資格名・発行日及び有効期限が分かるものを添付の上申請すること。  
※2024年12月から「Professional ChromeOS Administrator」のみシリーズIDの発行が廃止されたため、この資格のみシリーズIDがブランクのものを受け付けすることとする。  
４．証明書の取得に時間を要する場合（注）、資格発行者に問い合わせをし、  
合格者名及び合格が分かるものを添付の上、申請の仮受付としてワークフローで申請するものとし、これを申請とみなして受領するものとする。  
但し、仮受付された申請日から20日までの間に正式な証明書が添付出来ない場合は、  
資格取得期間が確認出来ないため申請を却下とする。  
なお、仮受付された申請日から20日までの間に正式な証明書が発行された場合、  
仮受付された申請を自身で差し戻し、証明書を添付の上再申請すること。  
その場合の申請は仮受付日を申請日とする。

注）証明書の取得に時間を要する場合とは、当初予定された発行期日になっても発行されていない証明書を入手するために問い合わせなどの時間を要する場合をいい、自己の過失（紛失等）で申請期日を間に合わせるために行った申請の仮受付は認められないものとする  
また、資格発行者に確認した証明書発行が遅れた理由を問い合わせる場合がある。

１項については2019年8月9日支給給与から適用開始とする。  
２項については2019年6月以前より変更ないものとする。  
３項については2023年11月24日支給給与から適用開始とする。  
４項については2023年11月30日から適用開始とする。

# ③休業・休職中の資格新規取得・更新について

１．休業とは産前休業、産後休業、育児休業、介護休業のことを指す。  
　　休職とは休業以外の理由で会社を休むことを指す。  
２．休業・休職中に資格を取得又は更新した場合の資格取得申請時期について  
　　復職後に資格取得申請をする。  
　　（休業・休職中は原則会社アカウントを使用できません）  
３．資格手当の支給開始時期及び支給期間について  
　　・休業の場合は、各月月末までに申請された資格取得申請に対して翌月25日から支給開始となる。  
　　・休職の場合は、各月月末までに申請された資格取得申請に対して翌々月25日から支給開始となる。  
　　・支給期間は「資格有効期間の残余の期間」となる。  
４．お祝い金について  
　　①お祝い金 に則り支給する。  
　　なお、休業・休職中に資格取得をし、また休業・休職中にその資格の有効期間が切れた場合は、その資格の更新を行い、復職後、同時に新規と更新の資格取得申請をする場合に限りお祝い金を支給する。  
５．受験費用補助について  
　　・発生から精算までにその年の会計年度（12月31日）を超えない場合は、復職後本人が経費精算  
　　・発生から精算までにその年の会計年度（12月31日）を超える場合は、労務代理申請となるため、領収証を労務へ提出
