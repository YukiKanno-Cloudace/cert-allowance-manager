# 資格手当管理システム 仕様書

作成日: 2025年12月25日

## 1. 資格手当制度の概要

### 1.1 制度の目的

従業員のスキル向上と専門知識の習得を奨励し、業務能力の向上を図ることを目的とした手当制度。

### 1.2 手当の種類

本制度には以下の2種類の手当があります。

#### 1.2.1 お祝い金

- 資格取得時に一度のみ支給
- 初回合格時のみ対象（更新時は対象外、入社前取得資格も対象外）
- Google Cloud資格の場合、各資格で申請が早い順に5名まで追加のお祝い金5万円を支給
  - ただし、以下の3資格は追加お祝い金の対象外：
    - Professional Cloud Architect
    - Professional Data Engineer
    - Associate Cloud Engineer

#### 1.2.2 月額手当

- 資格取得後、毎月継続して支給
- 複数資格を取得した場合は合算して支給
- **月額手当の支給上限：10万円**

### 1.3 支給タイミング

#### お祝い金

- 各月20日までの申請 → 翌月25日に支給
- 各月21日以降の申請 → 翌々月25日に支給

#### 月額手当

- 各月末日までの申請 → 翌々月25日から支給開始

### 1.4 その他重要事項

- 資格試験の受験料は各資格1回分のみ支給
- 資格更新時の受験料も1回分のみ支給対象
- 資格取得のための受験は業務時間外とする

## 3. 資格の有効期限詳細

### 3.1 Google Cloud 認定資格

#### Foundational / Associate レベル（有効期限3年）

- **Cloud Digital Leader**：3年間
- **Generative AI Leader**：3年間
- **Associate Cloud Engineer**：3年間
- **Associate Google Workspace Administrator**：3年間
- **Associate Data Practitioner**：3年間

#### Professional レベル（有効期限2年）

- **Professional Cloud Architect**：2年間
- **Professional Database Engineer**：2年間
- **Professional Cloud Developer**：2年間
- **Professional Data Engineer**：2年間
- **Professional Cloud DevOps Engineer**：2年間
- **Professional Cloud Security Engineer**：2年間
- **Professional Cloud Network Engineer**：2年間
- **Professional Machine Learning Engineer**：2年間
- **Professional Security Operations Engineer**：2年間

#### 更新方法

- **Foundational / Associate資格**：有効期限の180日前から再認定試験の受験が可能
- **Professional資格**：有効期限の60日前から再認定試験の受験が可能
- 再認定試験は、最新バージョンの試験に合格する必要がある
- 有効期限切れ前にメール通知と受験料50%割引クーポンが提供される
- 失効後30日以内であれば更新可能
- 一部の資格（Professional Cloud Architect、Professional Data Engineer、Associate Cloud Engineer、Cloud Digital Leader）は短縮版の更新試験が利用可能

**参考**: [Google Cloud 認定資格の更新（公式）](https://support.google.com/cloud-certification/answer/9907853)

### 3.2 Chrome OS 認定資格

- **Professional ChromeOS Administrator**：3年間
- **更新方法：** Google Cloud資格と同様の更新プロセス
- **備考：** 2024年12月よりシリーズIDの発行が廃止

### 3.3 CNCF 認定資格

- **有効期限：** 3年間
- **更新方法：** 有効期限内に再認定試験に合格する必要がある
- **対象資格：**
  - Certified Kubernetes Application Developer (CKAD)
  - Certified Kubernetes Administrator (CKA)
- **備考：** 有効期限の60日前から再認定試験の受験が可能

### 3.4 PMI 認定資格

- **有効期限：** 3年間（3年サイクル）
- **更新方法：** 3年間で60 PDU (Professional Development Units) の取得が必要
- **対象資格：**
  - PMP (Project Management Professional)
- **備考：** PDUは教育活動、実務経験、ボランティア活動などで取得可能

### 3.4 有効期限管理の注意点

- 資格の有効期限が切れた場合、月額手当の支給は停止される
- 資格を更新した場合、更新後も月額手当は継続して支給される
- 更新時の受験料も1回分のみ補助対象となる
- 更新時はお祝い金の支給対象外

## 4. 申請要件と注意事項

### 4.1 申請の定義

- 原則、不備のない申請のことを指す
- 本人過失の不備（合格証明書の添付忘れ等）がある申請は、申請とはみなさない
- 本人過失のない不備（Google側の問題等）の申請は、申請とみなす

### 4.2 証明書について

- 合格者名、合格した資格名、発行日、有効期限が分かるものを添付すること
- Professional ChromeOS Administratorのみ、シリーズIDがブランクのものを受け付ける（2024年12月からシリーズID発行廃止のため）

### 4.3 証明書の取得に時間を要する場合

- 資格発行者に問い合わせをし、合格者名および合格が分かるものを添付の上、仮受付として申請可能
- 仮受付日から20日以内に正式な証明書を添付して再申請すること
- 20日以内に正式な証明書が添付できない場合は申請却下
- 自己の過失（紛失等）による仮受付は認められない

### 4.4 休業・休職中の取扱い

- 休業：産前休業、産後休業、育児休業、介護休業
- 休職：休業以外の理由で会社を休むこと
- 資格取得申請は復職後に行う
- 手当支給開始時期：
  - 休業の場合：各月末日までの申請 → 翌月25日から支給開始
  - 休職の場合：各月末日までの申請 → 翌々月25日から支給開始
- 支給期間は資格有効期間の残余の期間となる

## 5. システム要件（アプリケーション開発向け）

### 5.1 管理すべきデータ

#### 資格マスタ

- 資格ID
- 資格名
- カテゴリ
- お祝い金額
- 月額手当額
- 有効期限（年数）
- 追加お祝い金対象フラグ（Google Cloud資格のみ）
- 追加お祝い金対象外フラグ（特定の3資格）
- ステータス（有効/無効）

#### 従業員資格取得情報

- 従業員ID
- 資格ID
- 取得日
- 有効期限日（取得日 + 資格の有効期限年数）
- 申請日
- お祝い金支給日
- 月額手当支給開始日
- 証明書ファイル
- シリーズID（該当する場合）
- 申請ステータス（申請中/承認/却下/仮受付）
- 更新フラグ（初回取得/更新）

#### 月額手当支給履歴

- 従業員ID
- 支給年月
- 支給資格リスト
- 支給額合計（上限10万円チェック）
- 支給ステータス

#### 追加お祝い金管理（Google Cloud資格）

- 資格ID
- 受付順位（1〜5位）
- 従業員ID
- 申請日時
- 追加お祝い金支給日

### 5.2 必要な機能

#### 基本機能

1. 資格取得申請機能
2. 申請承認/却下機能
3. お祝い金計算・支給管理機能
4. 月額手当計算・支給管理機能
5. 資格有効期限アラート機能
6. 追加お祝い金（Google Cloud）管理機能

#### アラート機能

- 資格有効期限が3ヶ月前になったらアラート
- 資格有効期限が1ヶ月前になったらアラート
- 資格有効期限が切れたら月額手当支給停止

#### レポート機能

- 従業員別資格取得状況
- 月別支給額集計
- 資格別取得者数
- 有効期限切れ間近の資格リスト

### 5.3 計算ロジック

#### お祝い金支給日計算

```
IF 申請日 <= 当月20日 THEN
  支給日 = 翌月25日
ELSE
  支給日 = 翌々月25日
END IF
```

#### 月額手当支給開始日計算

```
IF 申請日 <= 当月末日 THEN
  支給開始日 = 翌々月25日
END IF
```

#### 月額手当合計上限チェック

```
従業員の月額手当合計 = SUM(取得資格の月額手当)
IF 従業員の月額手当合計 > 100,000 THEN
  支給額 = 100,000
ELSE
  支給額 = 従業員の月額手当合計
END IF
```

#### 有効期限日計算

```
有効期限日 = 取得日 + 資格の有効期限年数
```
